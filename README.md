# 💳 AI-Powered Credit Risk Prediction System

## 📌 Overview
This project predicts customer delinquency risk using financial data such as income, credit utilization, and payment history. It combines machine learning with explainable AI to support better financial decision-making.

## 🚀 Features
- Predicts delinquency risk (High / Low)
- Shows probability score
- Explains key factors influencing prediction
- Interactive Streamlit web app
- Synthetic dataset simulating real financial behavior

## 🧠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

## 📊 How It Works
1. Input customer financial data  
2. Model predicts delinquency probability  
3. System explains contributing factors  
4. Suggests business actions  

## 💼 Business Impact
- Helps identify high-risk customers early  
- Enables proactive intervention strategies  
- Improves collections efficiency  
- Supports explainable AI in financial systems
